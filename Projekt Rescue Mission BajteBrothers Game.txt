💡 Naziv igrice

🛠 Korak 1: Izaberi najlakši alat (Engine)
Za početnike i mobilne igre najbolji su alati gdje ne moraš puno programirati kodom:
● GDevelop: Potpuno besplatan alat. Radi na principu "poveži kockice" (bez
koda). Odličan za 2D igre i direktno izbacuje igru za Android/iOS.
● Construct 3: Vrlo lagan, pokreće se u web pregledniku. Isto radi bez koda, na
principu vizualnih pravila.
🎮 Korak 2: Koncept i mehanika (Leveli)
Igra će biti 2D avantura (pogled sa strane ili odozgo) u kojoj dječaci voze bager ili
kombi.
● Level 1-3 (Skupljanje i potraga): Dječaci voze vozilo, skupljaju kanistere s
gorivom (Fuel) i bodove za dobrotu. Usput troše bodove da kupe tragove
(Hinweise) gdje su životinje skrivene.
● Glavna mehanika: Kada naiđu na kavez, ne koriste oružje. Koriste "alat za
dobro djelo" (npr. ključ ili bager) da oslobode životinje (medvjede, vukove,
ptice).
● Zadnji Level (Istiniti događaj): Umjesto životinje, u zadnjem levelu spašavaju
oteto dijete iz sumnjivog kombija. Blokiraju kombi bagerom (baš kao u pravoj
priči u Njemačkoj) i čekaju policiju.
📝 Korak 3: Unesi tvoju životnu poruku
Na kraju igrice, kada pobijede zadnji level i spase dijete, na ekranu se pojavljuje
medalja iz Njemačke i tvoja velika poruka:
"Bajtebrothers ne čini ništa iz osobne koristi. Najveća nagrada je učiniti
dobro djelo i pomoći onome tko je nemoćan. Sada ste i vi službeni
heroji!"

1. U GDevelop-u ili Construct-u klikni na Export (Izvoz).
2. Izaberi Android (APK).
3. Dobit ćeš datoteku koju preko kabela ili WhatsAppa prebaciš dječacima na
mobitel.
4. Instaliraju je i igraju igru kreiranu samo za njih.

# 📲 KAKO IGRU PRETVORITI U APK (Android)

1. U GDevelop klikni na **"Export"** (gornji desni kut)
2. Odaberi **"Mobile – Android (APK)"**
3. Popuni:
   - **App name:** BajteBrothers Rescue
   - **Package name:** com.bajtebrothers.rescue (jedinstveno)
   - **Version:** 1.0.0
4. Klikni **"Build APK"** (program šalje u oblak, treba 2-5 minuta)
5. Kad završi, klikni **"Download APK"**
6. Datoteka će se spremiti kao `bajtebrothers.apk` (oko 30-50 MB)

## Slanje na mobitel (Alex i Kevin)

- Opcija 1: Pošalji na WhatsApp → oni kliknu → instaliraju
- Opcija 2: Prebaci USB kabelom na Download folder
- Opcija 3: Uploadaj na Google Drive i podijeli link

## Instalacija na njihovom mobitelu

1. Kliknu na .apk datoteku
2. Mobitel kaže "Instalacija iz nepoznatih izvora nije dopuštena"
3. Kliknu **"Settings"** → dopuste instalaciju iz "WhatsApp" ili "File manager"
4. Vrate se i kliknu **"Instaliraj"**
5. Ikona se pojavi na ekranu – IGRAJU!

# 📲 KAKO IGRU PRETVORITI U APK (Android)

1. U GDevelop klikni na **"Export"** (gornji desni kut)
2. Odaberi **"Mobile – Android (APK)"**
3. Popuni:
   - **App name:** BajteBrothers Rescue
   - **Package name:** com.bajtebrothers.rescue (jedinstveno)
   - **Version:** 1.0.0
4. Klikni **"Build APK"** (program šalje u oblak, treba 2-5 minuta)
5. Kad završi, klikni **"Download APK"**
6. Datoteka će se spremiti kao `bajtebrothers.apk` (oko 30-50 MB)

## Slanje na mobitel (Alex i Kevin)

- Opcija 1: Pošalji na WhatsApp → oni kliknu → instaliraju
- Opcija 2: Prebaci USB kabelom na Download folder
- Opcija 3: Uploadaj na Google Drive i podijeli link

## Instalacija na njihovom mobitelu

1. Kliknu na .apk datoteku
2. Mobitel kaže "Instalacija iz nepoznatih izvora nije dopuštena"
3. Kliknu **"Settings"** → dopuste instalaciju iz "WhatsApp" ili "File manager"
4. Vrate se i kliknu **"Instaliraj"**
5. Ikona se pojavi na ekranu – IGRAJU!

🖼️ Kako da pripremiš slike – jednostavan način
Ne trebaš ništa posebno "upisivati" ili "uploadati" na Imgur. Evo šta trebaš uraditi:

Za svaku sliku (obje fotografije):
Klikni desnim tasterom miša na sliku koju želiš da koristimo

Odaberi opciju "Otvori sliku u novoj kartici" ili "Copy image address" (Kopiraj adresu slike)

Ako ti te opcije nisu dostupne, sačuvaj sliku na računalo (desni klik → "Save image as...")

Zatim uploadaj sliku na neki od ovih besplatnih servisa:

Servis	Kako se koristi
ImgBB (najlakši)	Otvori imgbb.com → klikni "Start Uploading" → odaberi sliku → klikni "Upload" → kopiraj "Direct link"
GitHub (sigurniji)	Napravi GitHub repo → uploadaj slike → commit → kopiraj raw link
Google Drive	Uploadaj sliku → podijeli je javno → kopiraj link koji završava sa export=view
Najtoplije preporučujem ImgBB – brz je i jednostavan, ne treba ti nalog.

<script>
    (function(){
        const canvas = document.getElementById('gameCanvas');
        const ctx = canvas.getContext('2d');
        
        // --- FOTOGRAFIJE su već postavljene: intro = slika1.jpg, secret = slika2.jpg
        
        // --- GAME STATE ---
        let player = { x: 450, y: 300, angle: 0, speed: 0, maxSpeed: 4, acc: 0.3, turnSpeed: 0.08 };
        let fuel = 100;
        let stars = 0;
        let selectedChar = "Alex"; 
        let gameActive = true;
        let levelComplete = false;
        let currentLevel = 1;
        let rescued = false;
        let cageUnlocked = false;
        
        // Items from shop
        let items = { goldenKey: false, superMagnet: false, speedBoost: false };
        
        // Level specific
        let trapavkoState = "idle";
        let trapavkoX = 650, trapavkoY = 400;
        let cageX = 500, cageY = 200;
        let animalEmoji = "🐺";
        
        // Level definitions
        const levels = {
            1: { name: "Šuma", animal: "Vuk", emoji: "🐺", trapTrick: "banana", mapColor: "#3c8c40", desc: "Spasite vuka!" },
            2: { name: "Planina", animal: "Jelen", emoji: "🦌", trapTrick: "snowman", mapColor: "#8cb8e8", desc: "Spasite jelena!" },
            3: { name: "Grad", animal: "Orao", emoji: "🦅", trapTrick: "bucket", mapColor: "#a0a0a0", desc: "Spasite orla!" },
            4: { name: "Finale", animal: "Dijete", emoji: "👶", trapTrick: "chase", mapColor: "#5a5a5a", desc: "Blokirajte kombi i spasite dijete!" }
        };
        
        let fuelBarrels = [];
        let starsList = [];
        
        function resetLevelCollectibles() {
            fuelBarrels = [];
            starsList = [];
            for(let i=0; i<3; i++) fuelBarrels.push({x: 150 + Math.random() * 600, y: 100 + Math.random() * 400, active: true});
            for(let i=0; i<5; i++) starsList.push({x: 100 + Math.random() * 700, y: 80 + Math.random() * 450, active: true});
        }
        
        function setupLevel(levelNum) {
            const level = levels[levelNum];
            if(!level) return;
            rescued = false;
            cageUnlocked = false;
            levelComplete = false;
            gameActive = true;
            cageX = 200 + Math.random() * 500;
            cageY = 100 + Math.random() * 400;
            trapavkoX = 100 + Math.random() * 700;
            trapavkoY = 100 + Math.random() * 400;
            player.x = 100;
            player.y = 300;
            player.speed = 0;
            animalEmoji = level.emoji;
            trapavkoState = "idle";
            if(levelNum === 4) {
                trapavkoX = 750;
                trapavkoY = 300;
                cageX = 850;
                cageY = 300;
            }
            resetLevelCollectibles();
            updateUI();
        }
        
        function updateUI(){
            document.getElementById('fuelVal').innerText = Math.floor(fuel);
            document.getElementById('starVal').innerText = stars;
            document.getElementById('charName').innerText = selectedChar;
        }
        
        // Joystick
        let stickActive = false;
        let stickX = 0, stickY = 0;
        const joyZone = document.getElementById('joystickZone');
        let actionPressed = false;
        
        function handleStart(e) { e.preventDefault(); stickActive = true; updateStick(e); }
        function updateStick(e) {
            const rect = joyZone.getBoundingClientRect();
            let clientX, clientY;
            if(e.touches) { clientX = e.touches[0].clientX; clientY = e.touches[0].clientY; }
            else { clientX = e.clientX; clientY = e.clientY; }
            let dx = clientX - (rect.left + rect.width/2);
            let dy = clientY - (rect.top + rect.height/2);
            let len = Math.hypot(dx,dy);
            let maxR = rect.width/2;
            if(len > maxR) { dx = dx/len * maxR; dy = dy/len * maxR; }
            stickX = dx / maxR;
            stickY = dy / maxR;
        }
        function handleEnd() { stickActive = false; stickX=0; stickY=0; }
        joyZone.addEventListener('touchstart', handleStart);
        joyZone.addEventListener('touchmove', updateStick);
        joyZone.addEventListener('touchend', handleEnd);
        joyZone.addEventListener('mousedown', handleStart);
        window.addEventListener('mousemove', (e) => { if(stickActive) updateStick(e); });
        window.addEventListener('mouseup', handleEnd);
        
        const actionBtn = document.getElementById('actionBtn');
        actionBtn.addEventListener('touchstart', (e) => { e.preventDefault(); actionPressed=true; setTimeout(()=>actionPressed=false,150); });
        actionBtn.addEventListener('mousedown', () => { actionPressed=true; setTimeout(()=>actionPressed=false,150); });
        
        // Game logic
        function updateMovement(){
            if(!gameActive || levelComplete) return;
            let move = stickActive ? { x: stickX, y: stickY } : { x:0, y:0 };
            let turn = move.x;
            let throttle = -move.y;
            if(throttle !==0) player.angle += turn * player.turnSpeed;
            let speedChange = throttle * player.acc;
            let maxSpd = player.maxSpeed;
            if(items.speedBoost && selectedChar==="Alex") maxSpd *= 1.3;
            player.speed += speedChange;
            if(player.speed > maxSpd) player.speed = maxSpd;
            if(player.speed < -maxSpd/2) player.speed = -maxSpd/2;
            player.speed *= 0.98;
            let dx = Math.sin(player.angle) * player.speed;
            let dy = -Math.cos(player.angle) * player.speed;
            player.x += dx;
            player.y += dy;
            player.x = Math.min(850, Math.max(50, player.x));
            player.y = Math.min(550, Math.max(50, player.y));
            
            if(Math.abs(player.speed) > 0.2){
                let consumption = 0.3;
                if(selectedChar === "Kevin") consumption *= 0.8;
                fuel = Math.max(0, fuel - consumption/10);
                if(fuel <= 0) gameActive = false;
            }
            
            for(let b of fuelBarrels) if(b.active && Math.hypot(player.x-b.x, player.y-b.y) < 35) { b.active=false; fuel=Math.min(100,fuel+30); stars+=5; updateUI(); playSound('collect'); }
            for(let s of starsList) if(s.active && Math.hypot(player.x-s.x, player.y-s.y) < 30) { s.active=false; stars+=10; updateUI(); playSound('collect'); }
            
            let level = levels[currentLevel];
            if(level && !rescued) {
                if((level.trapTrick === 'banana' || level.trapTrick === 'snowman' || level.trapTrick === 'bucket') && trapavkoState === "idle" && !cageUnlocked) {
                    if(actionPressed && Math.hypot(player.x-trapavkoX, player.y-trapavkoY) < 60) {
                        trapavkoState = "slip";
                        cageUnlocked = true;
                        stars += 20;
                        updateUI();
                        playSound('slip');
                        setTimeout(()=>{ trapavkoState="idle"; }, 1500);
                    }
                }
                else if(level.trapTrick === 'chase' && !rescued) {
                    if(trapavkoState === "idle" && !cageUnlocked) {
                        let dx = cageX - trapavkoX, dy = cageY - trapavkoY, dist = Math.hypot(dx,dy);
                        if(dist > 5) { trapavkoX += (dx/dist)*2; trapavkoY += (dy/dist)*2; }
                        if(Math.hypot(player.x-cageX, player.y-cageY) < 50) { cageUnlocked = true; stars += 30; updateUI(); playSound('cheer'); }
                    }
                }
            }
            
            if(cageUnlocked && !rescued && Math.hypot(player.x-cageX, player.y-cageY) < 50 && actionPressed){
                rescued = true;
                stars += 30;
                updateUI();
                playSound('cheer');
                levelComplete = true;
                gameActive = false;
                setTimeout(()=>{
                    if(currentLevel === 4) showEnding();
                    else { currentLevel++; setupLevel(currentLevel); }
                }, 1000);
            }
        }
        
        function draw(){
            const level = levels[currentLevel];
            ctx.fillStyle = level ? level.mapColor : "#3c8c40";
            ctx.fillRect(0,0,900,600);
            ctx.fillStyle = "#8b7a6b"; ctx.fillRect(0,280,900,40);
            ctx.fillStyle = "#e9c891"; ctx.fillRect(0,290,900,20);
            
            ctx.fillStyle = "#aa3333";
            fuelBarrels.forEach(b=>{ if(b.active){ ctx.fillRect(b.x-10,b.y-15,20,30); ctx.fillStyle="#cc5555"; ctx.fillRect(b.x-6,b.y-5,12,10); ctx.fillStyle="#aa3333"; } });
            ctx.fillStyle = "#ffdd77";
            starsList.forEach(s=>{ if(s.active){ ctx.beginPath(); for(let i=0;i<5;i++){ let ang=i*Math.PI*2/5; let x=s.x+Math.cos(ang)*10, y=s.y+Math.sin(ang)*10; if(i===0) ctx.moveTo(x,y); else ctx.lineTo(x,y); } ctx.fill(); } });
            
            ctx.fillStyle = trapavkoState === "slip" ? "#8b5a2b" : "#cc8844";
            ctx.fillRect(trapavkoX-20,trapavkoY-20,40,40);
            ctx.fillStyle = "#ffffff";
            ctx.fillRect(trapavkoX-8,trapavkoY-8,6,6);
            ctx.fillRect(trapavkoX+2,trapavkoY-8,6,6);
            
            ctx.fillStyle = "#a57c4c";
            ctx.fillRect(cageX-20,cageY-25,40,50);
            ctx.fillStyle = "#5a3a1a";
            ctx.fillRect(cageX-15,cageY-15,30,30);
            if(!rescued) { ctx.fillStyle="#fff"; ctx.font="bold 14px monospace"; ctx.fillText(animalEmoji,cageX-8,cageY); }
            else { ctx.fillStyle="green"; ctx.fillText("✅ SPASEN",cageX-25,cageY-30); }
            
            ctx.save();
            ctx.translate(player.x, player.y);
            ctx.rotate(player.angle);
            ctx.fillStyle = selectedChar==="Alex"?"#e8b13c":(selectedChar==="Kevin"?"#3c6ea8":"#6ca84c");
            ctx.fillRect(-20,-15,40,30);
            ctx.fillStyle = "#333";
            ctx.fillRect(-10,-8,20,16);
            ctx.restore();
            
            ctx.fillStyle = "#222";
            ctx.fillRect(20,20,200,20);
            ctx.fillStyle = fuel>50?"#2eff2e":"#ff6666";
            ctx.fillRect(20,20,fuel*2,20);
            ctx.fillStyle = "white";
            ctx.fillText(`⭐ ${stars}`, 20, 65);
            if(level && !rescued) ctx.fillText(level.desc, 20, 100);
        }
        
        function playSound(type){
            let audioCtx = new (window.AudioContext || window.webkitAudioContext)();
            let osc = audioCtx.createOscillator();
            let gain = audioCtx.createGain();
            osc.connect(gain);
            gain.connect(audioCtx.destination);
            osc.type = "sine";
            let now = audioCtx.currentTime;
            if(type==='collect'){ osc.frequency.value=880; gain.gain.setValueAtTime(0.2,now); gain.gain.exponentialRampToValueAtTime(0.0001,now+0.3); osc.start(); osc.stop(now+0.2);}
            if(type==='slip'){ osc.frequency.value=220; gain.gain.setValueAtTime(0.3,now); osc.start(); osc.stop(now+0.2);}
            if(type==='cheer'){ osc.frequency.value=1318.52; gain.gain.setValueAtTime(0.2,now); osc.start(); osc.stop(now+0.5);}
            audioCtx.resume();
        }
        
        function showShop() { document.getElementById('shopStars').innerText = stars; document.getElementById('shopScreen').style.display = 'flex'; }
        function hideShop() { document.getElementById('shopScreen').style.display = 'none'; }

            else { alert("Nema dovoljno zvjezdica ili ste već kupili!"); return; }
            updateUI();
            document.getElementById('shopStars').innerText = stars;
        }
